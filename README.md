# Анализ фильмов с сайта Letterboxd

## Задача

Заказчик — режиссёр, который хочет понять, какие характеристики фильма (жанр, страна производства, язык, период выпуска, наличие «Оскара») чаще встречаются у высоко оценённых фильмов.

Цель проекта — собрать датасет популярных фильмов с Letterboxd и исследовать связи между этими характеристиками и рейтингом. Проект ищет **корреляции, а не причинно-следственные связи** — задача описательная, а не предсказательная.

### Проверяемые гипотезы

- Фильмы с «Оскаром» в среднем имеют более высокий рейтинг
- Производство с участием США чаще встречается среди высокорейтинговых фильмов
- Фильмы только на английском отличаются по рейтингу от многоязычных
- Международные проекты (несколько стран) отличаются по рейтингу
- Жанры и период выпуска связаны с рейтингом

## План работы

1. Парсинг списка популярных фильмов с [letterboxd.com/films/popular](https://letterboxd.com/films/popular/) и сбор деталей по каждому фильму
2. Очистка данных: пропуски, дубликаты, приведение типов
3. Создание новых признаков: временные, языковые, производственные, награды
4. Разведывательный анализ: сравнение средних, группировки, корреляции
5. Визуализация ключевых результатов и формулировка выводов

## Источники данных

- **Letterboxd** ([letterboxd.com/films/popular](https://letterboxd.com/films/popular/)) — основной список популярных фильмов и детали по каждому (актёры, режиссёр, страна, язык, студия, жанр)
- **Rotten Tomatoes** — справочник фильмов-победителей в номинации «Лучший фильм» (для признака `has_oscar`)

Собрано более 2000 фильмов.

## Структура данных

**df_main** — название, год, ссылка, рейтинг

**details** — актёры, режиссёр, страна, язык, студия, жанр

**dict_oscar** — справочник фильмов с премией «Оскар»

**full_film_merged** (итоговый датафрейм) включает все исходные поля плюс созданные признаки:

| Признак | Описание |
|---|---|
| has_oscar | Признак наличия премии «Оскар» |
| decade / decade_start | Десятилетие выпуска |
| century | Век выпуска |
| five_year | Пятилетие выпуска |
| hollywood | Признак съёмки в Голливуде |
| international | Признак участия нескольких стран в производстве |
| english_only | Признак фильма только на английском языке |
| n_languages | Количество языков в фильме |
| movie fees | Кассовые сборы (см. ограничения) |
| genres_split | Жанры, разбитые по отдельным значениям |

## Предобработка данных

- Извлечение года выпуска из строки «название (год)» через регулярные выражения
- Объединение нескольких датафреймов по ключу — ссылке на фильм
- Удаление дубликатов по уникальному идентификатору (URL)
- Приведение списковых значений (актёры, жанры, языки) к строковому формату
- Создание бинарного признака `has_oscar` на основе справочника наград
- Устранение дублирования языков через приведение к множеству (`language_set`)
- Пропуски в категориальных признаках сознательно оставлены незаполненными, чтобы не искажать реальные данные

### О признаке кассовых сборов

`movie fees` — это **приближённая симуляция**, построенная на основе исторических усреднённых данных по десятилетиям с произвольным уровнем инфляции (3%) и случайной генерацией в разумном диапазоне. Признак создан в иллюстративных целях для расширения количественных данных в датасете и демонстрации инструментов анализа — **делать содержательные бизнес-выводы по нему нельзя**.

## Ключевые результаты

### Корреляционный анализ (коэффициент Спирмена)

- Временные признаки (год, пятилетие, век) показывают умеренную отрицательную корреляцию с рейтингом (-0.41) — более новые фильмы в выборке имеют немного более низкий рейтинг
- `has_oscar` и `n_languages` не показывают значимой корреляции с другими признаками
- `movie_fees` коррелирует с временными признаками умеренно положительно, но это искусственный признак

### Динамика по векам и десятилетиям

- Фильмы XX века в среднем имеют более высокие и компактно распределённые рейтинги
- В XXI веке больше разброс оценок и больше низко оценённых фильмов — вероятно, из-за роста объёма выпускаемого контента
- Сам факт века выпуска не определяет качество фильма — высокорейтинговые картины встречаются в обоих периодах

### Язык и производство

- Подавляющее большинство популярных фильмов снято на одном языке, особенно заметно в 2000–2020-х
- Доля многоязычных фильмов растёт, но английский остаётся доминирующим
- Если фильм снят в США — рейтинг в среднем ниже, чем у фильмов, снятых не в США (при большом стандартном отклонении в обеих группах)

### Жанры

Самый стабильный во времени жанр — **драма**. В последние 5–10 лет растёт количество фильмов в жанрах ужасы, приключения, комедия и экшн, а в последние 5 лет — триллер.

### «Оскар» и рейтинг

Фильмы с премией «Оскар» имеют более высокий средний рейтинг и меньшее стандартное отклонение — они сосредоточены в зоне повышенных оценок. При этом связь не является определяющей: среди высокорейтинговых фильмов много картин без наград.

## Главные выводы

1. Жанровая структура популярных фильмов стабильна во времени — драма остаётся самым устойчивым жанром
2. Премия «Оскар» связана с более высоким рейтингом, но не гарантирует его
3. Языковые и производственные признаки слабо влияют на рейтинг
4. Корреляции между рейтингом и количественными признаками в целом слабые — однозначные выводы о факторах успеха фильма сделать нельзя на основе только этих данных

## Ограничения

- Выборка ограничена популярными фильмами Letterboxd и не репрезентативна для всего кинематографа
- Признак кассовых сборов сгенерирован искусственно и не подходит для содержательного анализа — в будущем стоит заменить реальными данными о бюджете
- Проект ограничен описательной статистикой и поиском корреляций; причинно-следственные связи не исследовались
